The missing Netbeans code formatter for x-sql.

Did you ever try to reformat your SQL on Netbeans and nothing happened?

This plugin adds a SQL reformatting task to our favorite IDE. Just install it via it's own Update Center or by downloading the NBM.

After that it works like it should:

After reformatting:

The plugin uses a webservice to reformat the code. It uses a fork of the Github Project sqlparse from @andialbrecht. We forked the project and host the service for free use on a cismet host. Feel free to fork our or Andi's project to host your own formatting service. Just change the URL in the plugin settings to your own service and you're done.

If you see room for improvement, just file an issue or fork the project and send us a pull-request. We would love to collaborate.
